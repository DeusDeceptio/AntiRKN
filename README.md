# AntiRKN
<dl>
  <dd>Приложение преднозначено для небольшой настройки, запуска zapret`a без использования проводника, так же есть возможность подкреить другие обходы и запускать прямо из приложения.</dd>
</dl>

## Как использовать
<dl>
  <dd>Для работы необходимо закинуть .exe файл приложения в папку zapret`a. В других сборка могут быть другие файлы, важно чтобы .exe файл был в одной папке с папкой "bin". Должна получиться следующая структура в проводнике:</dd>

## Функционал
<p align="center" width="100%">
<img width="784" height="611" alt="image" src="https://github.com/user-attachments/assets/08812ea7-b34d-4651-8da4-c466dbe7b943" /></p>

- Окно "Конфиги Zapret" позволяет выбрать и запустить нужный вам конфиг, при необходимости запомнить конфиг по умолчанию и запускать его одной кнопкой. Конфиг по умолчанию так же будет отображён в трее;
- Окно "Выбор листов для Zapret" позволяет выбирать необходимые списки адресов для обхода (работает если ваша сборка поддерживается это);
- Окно "Прочие обходы" позволяет добавить другие обходы и VPN для запуска из приложения или трея;
- "Тихий режим" - сворачивает приложение в трей, без отркрытия основного окна. После главное окно можно открыть двойным кликом по значку;
- "Остановить все" - отключает все добавленные обходы;
- "Обновить" - обновляет интерфейс и списки;
- "Сбросить" - сбрасывает настройки до значений по умолчанию, список добавленных обходов так же будет отчищен;
- Выход осуществляется через кнопку "Выход". На ✕ приложение скрывается в трей;
- Кнопки "Авто-поиск конфига" и "Автозапуск конфига" доступны только со сборкой рекомендованной в конце.

Нажав ПКМ по иконке в трее вы можете: 

- открыть главное окно приложения
- запустить конфиг zapret`a по умолчаню;
- будут отображены добавленные вами обходы;
- закрыть все запущенные обходы;
- закрыть данное приложение.
